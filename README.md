# DIO-Fundamentos_Cloud_AWS
Repositório contendo os desafios e anotações do curso de Fundamentos de Cloud da AWS da plataforma DIO

##############################################################################################################################################

DESAFIO 1:

Esse desenho representa uma arquitetura simples na Amazon Web Services usando uma instância Amazon EC2 conectada a volumes Amazon EBS e a um banco Amazon RDS.

Fluxo geral:

1) O usuário acessa a aplicação
2) A aplicação envia um arquivo para o volume EBS E:
3) A instância Amazon EC2 acessa esse EBS para ler/processar o arquivo
4) Depois disso, a EC2 salva informações no Amazon RDS
5) A EC2 também usa outro volume Amazon EBS F: para armazenamento adicional
